# twitchdashboard

> a simple twitch dashboard
[Temp Demo](https://flavz.github.io/twitchdashboard/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# deploy to github Pages  
npm run deploy
```
